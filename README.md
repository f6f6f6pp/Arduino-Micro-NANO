# 零件表
-ATMega32U4 X1  
-0603 22Ω X2 [位置在USB口下方兩個電阻位]  
-0603 330Ω X2 [位置在RX/TX LED 下方兩個電阻位]  
-0603 1uF X1 [C1]  
-0603 LED X2 [位置在USB口左方RX/TX位，靠近電阻為正極，靠近USB口為負極]  
-16Mhz 晶震 X1 [規格為 3215]  
-SMD Micro-USB 母座 X1  
-7P 針腳 *2  
# 注意事項
燒錄 BootLoader 時，請選擇燒錄 Arduino Micro 的 BootLoader  
RX 腳位是 D0  
TX 腳位是 D1  
RX/TX 腳位使用不影響燒錄，請放心
